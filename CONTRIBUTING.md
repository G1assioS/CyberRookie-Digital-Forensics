# Contributing

Contributions should be educational, technically accurate, and safe.

## Accepted Contributions

- Corrections to lesson notes
- Safe lab improvements
- Open-source forensic scripts
- Sanitized CTF write-ups
- Public dataset references
- Tool documentation updates

## Requirements

- Use only authorized or public evidence.
- Do not include credentials, personal data, or private case material.
- Do not publish solutions for active competitions.
- Explain commands and findings clearly.
- Cite external sources when appropriate.
- Test scripts before submitting them.
