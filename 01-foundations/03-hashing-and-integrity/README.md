## Planned Learning Objectives
- Understand cryptographic hashes
- Compare MD5, SHA-1, and SHA-256
- Create and verify hashes
- Understand original evidence and working copies
- Detect evidence modification

## Planned Lab
Create a safe text evidence file, record its SHA-256 hash, create a working copy, modify the copy, and verify that the original evidence remains unchanged.
