# CyberRookie Digital Forensics

A structured public learning repository for **Digital Forensics, CTF Forensics, and Professional DFIR**.

This repository supports the Digital Forensics track published on the main **CyberRookie** YouTube channel. It contains lesson notes, hands-on labs, commands, scripts, CTF write-ups, professional-style case studies, report templates, and supporting resources.

## Who This Repository Is For

- Beginners learning Digital Forensics
- CTF players interested in Forensics challenges
- Students preparing for DFIR careers
- SOC and Incident Response learners
- Aspiring Digital Forensic Examiners
- Viewers following CyberRookie videos

## Learning Tracks

### Digital Forensics: Zero to Advanced

- Evidence preservation and hashing
- File systems and storage
- Windows, Linux, and macOS Forensics
- Memory, Network, and Email Forensics
- Mobile Forensics
- Multimedia, Steganography, Documents, and Databases
- Malware Forensics and Incident Response
- Cloud Forensics and advanced specializations

### CTF Forensics

- Metadata and file signatures
- Corrupted files and file carving
- Image, Audio, and Video Forensics
- Steganography
- Disk images and memory dumps
- PCAP analysis
- Browser, mobile, log, and database challenges
- Forensic scripting and automation

### Professional DFIR

- Evidence acquisition and verification
- Endpoint investigations
- Phishing and email attribution
- Insider threats and ransomware
- Memory and network investigations
- Mobile and cloud incidents
- Timeline reconstruction
- Professional forensic reporting

## Ethics and Authorization

All practical work must use self-created evidence, public training datasets, authorized lab environments, CTF challenge files, or evidence for which explicit permission has been granted.

Do not publish private evidence, credentials, personal information, client data, active competition solutions, or unauthorized investigation material.

## Evidence Files

Large forensic evidence files are not stored directly in this repository. Lessons will link to trusted public datasets or provide instructions for creating safe training evidence.

## Language

Explanations are primarily created for Bangla-speaking learners, while technical terms, commands, filenames, and documentation remain in English.
